# Flask-tutorial
Tutorial flask
# Use the package manager [pip](https://pip.pypa.io/en/stable/) to install flask.
```bash
pip install flask
```

install the file
```bash
pip install flaskr-1.0.0-py3-none-any.whl
```


Run init-db to create the database in the instance folder.

```bash
flask --app flaskr init-db
```
